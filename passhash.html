How to Configure Your Network Connection

1 Enter your Eduroam password to generate the MD4 hash. For other networks, enter your password directly in the configuration.
2 The generated hash will automatically replace the HASH value in the configuration template.
3 When using AP=eduroam, your ID must be a valid email address. For other networks, any ID is acceptable.
4 Click "Download wireless.txt" or "Download 50-cloud-init.yaml" to save your configuration file after validation passes.

About Network Configuration

Eduroam (education roaming) is a secure, worldwide roaming access service developed for the international research and education community.

Your configuration requires:

  • • ID: For Eduroam, use your institutional email. For other networks, any identifier.
  • • HASH: MD4 hash of your password in UTF-16LE format (Eduroam only)
  • • AP: Set to "eduroam" for Eduroam networks, or your network name for others
